# Quick Start

Get started with Spectral-Hub in minutes.

## Installation

Install the package using pip:

```bash
pip install specbench
```

## Basic Usage

### Command Line Interface

```bash
# Show help
specbench --help

# Run evaluation
specbench eval --model your-model --dataset your-dataset
```

### Python API

```python
import specbench

# Basic usage
result = specbench.hello()
print(result)  # Output: Hello from Specbench!
```
